# How to Contribute

1. Fork the repository
2. Add file in correct folder
3. Use proper naming:
   subject_year_semester_type.pdf
4. Submit Pull Request

#*For the professional Elite who's using git:

#1.git clone https://github.com/Alialalaa/CSutmb
#2.cd CSutmb
#3.*Add the Original Repository as Upstream (Important) This keeps your fork updated:
#3.git remote add upstream https://github.com/OWNER_USERNAME/CSutmb.git
#4.git checkout -b example-add-math-exam-2024
#5.# add your files
#6.git add .
#7.git commit -m "Add file"
#8.git push origin add-file-branch

#*Create a Pull Request
#1.Go to your fork on GitHub.
#2.Click Compare & Pull Request.
#3.Write a short description.
#4.Click Create Pull Request.

#Done ✅
