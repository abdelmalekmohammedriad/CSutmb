# 📚 UTMB Computer science engineering path 

An open library of exams, TDs, TPs and TESTs.

## Structure
- 5 Years
- 2 Semesters per year
- TD / TP / TEST / EXAM

## How to Contribute

1. Fork the repository
2. Add file in correct folder
3. Use proper naming:
   subject_year_semester_type.pdf
4. Submit Pull Request

*For the professional Elite who's using git:

-git clone https://github.com/Alialalaa/CSutmb
-cd 
*Add the Original Repository as Upstream (Important) This keeps your fork updated:
-git remote add upstream https://github.com/OWNER_USERNAME/CSutmb.git
-git checkout -b example-add-math-exam-2024
-# add your files
-git add .
-git commit -m "Add file"
-git push origin add-file-branch

*Create a Pull Request
Go to your fork on GitHub.
Click Compare & Pull Request.
Write a short description.
Click Create Pull Request.

Done ✅
